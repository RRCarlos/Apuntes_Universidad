# Apuntes_Universidad

Apuntes universitarios organizados por asignatura.

## Asignaturas

### Gramática española
Apuntes de gramática española: fonética y fonología, lingüística, morfología y sintaxis.

### Historia Contemporánea
Apuntes de Historia Contemporánea: revoluciones, restauración, imperialismo, fascismo y nazismo.

### Historia Moderna de España
Apuntes de Historia Moderna de España: bloques temáticos I-IV orientados a preguntas de examen.
